Caffeinated.Beanstalk
=====================

A .NET library for using [Beanstalk][beanstalk].

There are other libraries, but they seem to have been abandoned:

* [libBeanstalk.NET][libbeanstalk]
* [Beanstalk-Sharp][beanstalk-sharp]
